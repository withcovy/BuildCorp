# 협업 시스템

## 3가지 협업 레벨

### 1. 내 여러 회사 간 협업

한 사용자가 여러 회사(프로젝트)를 운영할 때:
- 회사 A의 에이전트가 회사 B에 업무 요청
- 프로젝트 간 데이터/결과물 공유
- 로컬에서 처리 (네트워크 불필요)

### 2. 멀티플레이어 (다른 사용자 간)

- WebSocket (Socket.io) 기반 협업 서버
- **"외주" 개념**: 다른 사용자 회사의 에이전트에게 업무 위탁
- 결과물 주고받기
- 다른 사용자의 회사와 연결

### 3. 외부 서비스 연동

MCP를 통해 외부 도구와 연결:
- **Slack**: 에이전트가 메시지 전송/수신
- **GitHub**: 코드 PR 생성, 이슈 관리
- **Notion**: 문서 작성/업데이트
- **Google Drive**: 파일 저장/공유
- **커스텀 MCP 서버**: 뭐든 추가 가능

상세 내용은 [mcp.md](mcp.md) 참조.
