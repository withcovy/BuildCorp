# 인벤토리 시스템

## 컨셉

에이전트에게 부여하는 도구를 **RPG 인벤토리** 방식으로 관리.
도트 아이콘으로 도구를 표현하고, 드래그로 장착/해제.

## 구조

```
[장착 중]     → 에이전트가 현재 사용 가능한 도구
[보관함]      → 설치했지만 이 에이전트에겐 미배정
[MCP 스토어]  → 새 도구 검색/설치 (mcp.md 참조)
```

## 도구 권한 예시

에이전트 역할에 따라 도구를 다르게 배정:

```
개발 에이전트 "민수"
  ✅ readFile
  ✅ writeFile
  ✅ search
  ✅ runCommand
  ✅ GitHub (MCP)
  ❌ Slack (MCP)

검토 에이전트 "철수"
  ✅ readFile       ← 읽기만 가능
  ✅ search
  ❌ writeFile      ← 쓰기 권한 없음
  ❌ runCommand     ← 실행 권한 없음

마케팅 에이전트 "지영"
  ✅ readFile
  ✅ writeFile
  ❌ GitHub (MCP)   ← 필요 없음
  ✅ Slack (MCP)    ← 메시지 발송용
  ✅ Notion (MCP)   ← 문서 작성용
```

## 인터랙션

- **장착**: 보관함에서 장착 슬롯으로 **드래그**
- **해제**: 장착 슬롯에서 보관함으로 드래그
- **새 도구 추가**: MCP 스토어에서 검색/설치 → 보관함에 추가됨
- 도트 아이콘 스타일로 게임 느낌 유지
