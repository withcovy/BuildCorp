# 프로젝트 비전 & 정체성

## 한 줄 요약

> 도트 아트 가상 사무실에서 AI 직원들이 당신의 실제 업무를 수행하는 데스크톱 앱

## 정체성

- **게임이 아닌 실제 업무 도구** - 본질은 멀티 에이전트 AI 워크스페이스
- 게임적 요소(도트 디자인, 회사/직원/인벤토리)는 **시각화 수단**
- 사용자는 "사장", AI 에이전트들이 "직원"
- 직원들은 실제로 파일을 읽고, 코드를 짜고, 문서를 작성하는 등 진짜 업무를 수행

## 핵심 원칙

1. **실용성 우선** - 게임 재미보다 실제 업무 수행 능력이 핵심
2. **시각적 즐거움** - 도트 캐릭터들이 일하는 모습으로 진행 상황을 직관적으로 파악
3. **유연한 구조** - 어떤 업종이든 자유롭게 회사를 설정하고 팀을 구성
4. **협업 생태계** - 내 여러 회사 간, 다른 사용자 간, 외부 서비스와 연동

## 회사 구조

- 사용자가 자유롭게 회사 이름, 업종, 설명 설정
- 한 사용자가 **여러 회사(프로젝트) 운영 가능**
- 회사 내 **팀 자유 생성** (개발팀, 마케팅팀, 디자인팀 등)
- 팀 내 **AI 에이전트(직원) 배치**
- 각 회사는 독립적인 팀 구조 보유

## 업무 방식 3가지

| 방식 | 용도 | 설명 |
|------|------|------|
| **채팅** | 간단한 업무 | 에이전트 클릭 → 1:1 대화로 지시 |
| **태스크 보드** | 중간 복잡도 | 칸반 카드 배정 (Todo/In Progress/Review/Done) |
| **워크플로우** | 복잡한 다단계 | 에이전트 간 연계 파이프라인 구성 |

## 시간 시스템

- 실시간 동기화
- 업무 요청 → 결과 대기 → 확인의 자연스러운 흐름
