# UI 디자인

## 디자인 스타일

- **캐릭터**: 도트(픽셀) 아트
- **UI/배경**: HD 그래픽 (Tailwind CSS + shadcn/ui)
- **혼합 스타일**: 캐릭터는 도트, 나머지는 모던 HD

## 4개 메인 뷰

### 1. 사무실 뷰 (도트 아트)
- PixiJS 캔버스로 렌더링
- 팀별 구역에 도트 캐릭터 배치
- 상태별 애니메이션:
  - `idle`: 자리에 앉아 대기 💤
  - `working`: 타이핑 애니메이션 ⌨️
  - `break`: 커피머신 근처 휴식 ☕
  - `meeting`: 회의실에 모여있음 💬
- 워크플로우 진행 시 **직원 간 서류 전달 애니메이션**
- 캐릭터 클릭 → 채팅 패널 열림
- 사무실 가구/장비 커스터마이징

### 2. 대시보드 뷰 (HD UI)
- 회사 현황 카드 (활성 태스크, 완료율, 팀 상태)
- 에이전트 성과 통계
- 재정 현황

### 3. 태스크 뷰
- 칸반 보드 (Todo → In Progress → Review → Done)
- 태스크 카드 생성/배정
- 드래그&드롭으로 상태 변경

### 4. 워크플로우 뷰
- 직원 카드 기반 노드 에디터
- 실시간 진행 상태 상세 표시
- 템플릿 저장/불러오기
- 상세 내용은 [workflow.md](workflow.md) 참조
